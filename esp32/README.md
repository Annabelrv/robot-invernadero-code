# ESP32-CAM

Programa encargado de capturar imágenes, realizar un análisis cromático sencillo y enviar los resultados a Telegram.

## Hardware

* ESP32-CAM AI-Thinker

## Entorno

Desarrollado para **Arduino IDE** utilizando la placa:

> AI Thinker ESP32-CAM

## Funciones principales

* Inicialización de la cámara.
* Conexión a la red Wi-Fi.
* Captura de imágenes.
* Análisis cromático de la planta.
* Envío de fotografía y diagnóstico mediante un bot de Telegram.
* Comunicación con el robot mediante señales digitales.
