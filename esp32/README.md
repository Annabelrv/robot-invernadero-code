# ESP32-CAM
## Cámara, Análisis cromático y BOT de Telegram
Programa para la placa ESP32-CAM, incluyendo:
- comunicación con robot Keyestudio
- captura de foto
- analisis cromático para determinar diagnostico de planta
- conexión a BOT de Telegram para enviar resultado

### Configuración Arduino IDE
- **Placa**: AI Thinker ESP32-CAM
